# 🐍 Meu Aprendizado de Python | freeCodeCamp

Este repositório foi criado para centralizar e organizar todos os meus códigos, projetos, desafios e anotações teóricas desenvolvidos ao longo do curso de **Python** da [freeCodeCamp](https://www.freecodecamp.org/).

O principal objetivo é praticar a sintaxe da linguagem, lógica de programação e construir uma base sólida para futuros projetos em desenvolvimento de software, análise de dados ou automação.
